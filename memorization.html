Competitive Memory

Training to memorize a shuffled deck of 52 playing cards as fast as possible — the standard event in memory sports competitions like the USA Memory Championship and the World Memory Championships.

The key technique is the PAO system (Person–Action–Object): each of the 52 cards maps to a vivid person, a distinctive action, and a concrete object. Cards are encoded in groups of three using a memory palace — each locus in the palace holds one PAO triplet, giving 18 loci for a full deck. The brain remembers absurd scenes far better than abstract symbols.
